Switched to Dvorak? Hate the remapped shortcuts?

~ July 22, 2011 ~

I recently swiched to the Dvorak keyboard layout and have been loving it... except for one thing. All my keyboard shortcuts are whack! Most of them now require two hands, and I still accidentally use my QWERTY shortcuts quite often.

However, I'm happy to announce that I've found a solution! Using a free program called AutoHotKey, and a script I wrote, I'm now able to use my old QWERTY shortcuts even with a Dvorak layout. In effect it swiches my layout back to QWERTY only when the CTRL key is pressed. Feel free to use my script if you have been having the same frustrations!
