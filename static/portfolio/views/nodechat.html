The scope of NodeChat is tiny. It provides users with a single open, anonymous chatroom, and doesn't retain old chat messages. However, a few hidden features make NodeChat more complex than its stripped-down interface would have you believe.


Emoticons

NodeChat transforms special patterns into icons from the excellent Font Awesome icon library. Standard ASCII emoticons are mapped intuitively; inputting :) results in and <3 will display . More specialized icons can be accessed by enclosing the description of the icon in [square brackets]. For example, [like] will display as .

In addition to these standard emoticons, NodeChat is able to display any icon in the Font Awesome library by enclosing the icon's class inside [[double square brackets]] (i.e. [[fa-puzzle-piece]] shows up as ). For the curious, the implementation of the emoticon filter can be found here.


Profanity filter

One of the more entertaining pieces of NodeChat is its built-in profanity filter. NodeChat does its best to replace the most common dirty language with an innocuous translation, leading to entertaining sentences like "Mother pancaking piece of sheer beauty elbow mango smoothie!".
