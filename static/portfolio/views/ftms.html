FTMS (Field Trail Management System) is a rules-driven web application that enables DuPont contractors to manage data collected from chemical test trials.


Dates:
January 2015 - Current
Technologies:
Knockout, TypeScript, ASP.NET Web API 2, Entity Framework
Project type:
Work project

FTMS handles the complex interaction between several pre-existing applications within the DuPont ecosystem. FTMS is focused on two main concepts: trials and protocols. Protocols define how a particular chemical test is to be executed, while trials represent an implementation of a protocol. FTMS is tasked with ensuring that trial and protocol data is properly synced between users and applications. Chemicals processed through FTMS can be sold by DuPont with the confidence that they are safe to use and work as advertised.
